# make develop environment

## install node

### install fnm

curl -fsSL https://fnm.vercel.app/install | bash

### activate fnm

source ~/.bashrc

### download and install Node.js

fnm use --install-if-missing 22

## initialize npm

npm init -y

## install electron

npm install -D electron
